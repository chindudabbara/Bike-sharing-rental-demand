# Bike-sharing-rental-demand
* Constructed a machine learning model to predict bike-sharing rental demand using historical data and external factors including weather,    seasonality, holidays, and time-based features.
* Conducted exploratory data analysis (EDA), data preprocessing, and feature engineering to improve model accuracy and reliability.
* Benchmarked multiple models including Random Forest, Decision Tree, and Gradient Boosting using performance metrics RMSE, MAE, MSE, and     R2 Score.
* Deployed Random Forest Regressor as the final model achieving an R2 Score of 0.997, forecasting demand to improve bike availability         planning and reduce customer wait times.
